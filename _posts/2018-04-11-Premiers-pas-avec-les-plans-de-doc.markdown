Bien que les plans doc apparaissent toujours en tant que documents Word dans SharePoint, nous n'utilisons plus de modèles Word pour les créer.
généré automatiquement par l'application de base de données Create Doc Plan, ce qui nous permet de:
- Créez facilement des plans de documents sans formater le contenu
- Générer des rapports sur l'activité du plan de doc
- Générer automatiquement des documents de plan cohérents pour examen / approbation

### Créer un plan de doc

1. Sur la page SharePoint Planifier des documents, sélectionnez: [Créer un nouveau plan de doc](https://mcafee.sharepoint.com/sites/techpubs/Lists/Beta%20Test%20Doc%20plans/Item/newifs.aspx)
2. Entrez des informations sur votre nouveau plan de gestion de documents. L'entrée est requise uniquement pour les champs suivants: 
   - Titre: Un nom court pour votre plan de doc (exemple: nac-221-docplan). Veuillez ne pas utiliser de caractères de période (.) Dans le titre.
   - Sélectionnez le nom du produit: Votre produit doit figurer dans la liste Noms de produit de SharePoint.
   - Version de produit
   - Auteur du plan Doc
